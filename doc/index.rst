Swiftype Enterprise API Client
==============================

About
-----

Swiftype Enterprise API Client is the recommended way to interact with
Swiftype custom content sources through python.

The `source is on GitHub <https://github.com/swiftype/swiftype-enterprise-python>`_ and
the docs are on `ReadTheDocs <https://swiftype_enterprise.readthedocs.io/>`_.

Install with::

    $ python -m pip install swiftype_enterprise
